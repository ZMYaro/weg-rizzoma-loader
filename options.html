About

This extension makes it easy to add gadgets from the Wave Extensions Gallery to Rizzoma.

  1. Open the Rizzoma gadget menu.
  2. Type into the search box and click the search button or press Enter.
  3. The results from the gallery will show up in the Rizzoma menu.
  4. Click a gadget icon to add that gadget.

Note: There is a known issue that prevents selecting text in the search box with the mouse. For now, you will need to click in the search box and move the cursor with the arrow keys.
